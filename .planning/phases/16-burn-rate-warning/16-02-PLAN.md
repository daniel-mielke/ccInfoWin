    **Step 2: Create BurnRateNotificationService implementation**

    Create `CCInfoWindows/CCInfoWindows/Services/BurnRateNotificationService.cs`:
    - Namespace: `CCInfoWindows.Services`
    - Implements `IBurnRateNotificationService`
    - `using Microsoft.Windows.AppNotifications;` and `using Microsoft.Windows.AppNotifications.Builder;` and `using WinUI3Localizer;` and `using CCInfoWindows.Helpers;`
    - Private field: `private bool _notifiedBurnRate;`
    - `CheckBurnRate(BurnRatePrediction? prediction)`:
      - If `prediction == null`: set `_notifiedBurnRate = false` (reset cycle), return
      - If `_notifiedBurnRate`: return (already notified this cycle)
      - Set `_notifiedBurnRate = true`
      - Call `SendToast(prediction.MinutesUntilLimit)`
    - `private void SendToast(int minutesUntilLimit)`:
      - Guard: `if (!AppNotificationManager.IsSupported()) return;`
      - Format time label: call `BurnRateFormatter.FormatTimeLabel(minutesUntilLimit)` -- USES SHARED HELPER, NO LOCAL FORMATTING (DRY fix)
      - Get localized title: `Localizer.Get().GetLocalizedString("BurnRateNotificationTitle")`
      - Get localized body: `string.Format(Localizer.Get().GetLocalizedString("BurnRateNotificationBody"), timeLabel)`
      - Build notification:
        ```csharp
        var notification = new AppNotificationBuilder()
            .AddText(title)
            .AddText(body)
            .BuildNotification();
        notification.Tag = "usage-burnrate";
        AppNotificationManager.Default.Show(notification);
        ```
    - **IMPORTANT:** Do NOT add a `FormatTimeUntilLimit` method here. The formatting is handled entirely by `BurnRateFormatter.FormatTimeLabel` from Plan 01.
