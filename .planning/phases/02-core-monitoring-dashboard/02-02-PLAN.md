<task type="auto" tdd="true">
  <name>Task 2: ClaudeApiService with retry, caching, error handling, and DI registration</name>
  <files>
    CCInfoWindows/CCInfoWindows/Services/ClaudeApiService.cs,
    CCInfoWindows/CCInfoWindows/App.xaml.cs,

  <behavior>
    - FetchUsageAsync constructs URL with percent-encoded org ID
    - FetchUsageAsync includes Cookie: sessionKey={token} header
    - FetchUsageAsync includes anthropic-client-platform: web_claude_ai header
    - On 401: sends AuthStateChangedMessage(false), returns null, no retry
    - On 429/5xx: retries up to 2 more times with exponential backoff (1s, 2s)
    - On network error: retries up to 2 more times, then returns null
    - On success: deserializes JSON to UsageResponse, saves to cache, returns result
    - GetCachedUsage returns last in-memory cached response
    - SaveCacheAsync writes JSON to %LOCALAPPDATA%\CCInfoWindows\usage_cache.json
    - LoadCacheAsync reads from cache file, returns null if missing/corrupt
    - If org ID is null on fetch: attempts to fetch from /api/organizations first
    - ClaudeApiService registered as singleton in DI container
  </behavior>
  <action>
    **1. ClaudeApiService.cs** -- Implement IClaudeApiService:
    - Constructor takes `HttpClient`, `ICredentialService` via DI
    - Private fields: `_cachedUsage`, cache file path at `%LOCALAPPDATA%\CCInfoWindows\usage_cache.json`
    - `FetchUsageAsync`:
      1. Get sessionKey from `_credentialService.GetSessionToken()` -- if null, return null
      2. Get orgId from `_credentialService.GetOrganizationId()` -- if null, attempt migration (see below)
      3. Percent-encode orgId with `Uri.EscapeDataString(orgId)`
      4. Build URL: `https://claude.ai/api/organizations/{encodedOrgId}/usage`
      5. Create HttpRequestMessage with headers:
         - `Cookie: sessionKey={token}`
         - `anthropic-client-platform: web_claude_ai`
      6. Send request in retry loop (max 3 attempts):
         - On 401: send `WeakReferenceMessenger.Default.Send(new AuthStateChangedMessage(false))`, return null immediately (NO retry)
         - On 429 or 5xx (>= 500): wait `Task.Delay(attempt * 1000, ct)`, then retry
         - On network error (HttpRequestException): wait and retry
         - On success: deserialize with `JsonSerializer.DeserializeAsync<UsageResponse>`, save to `_cachedUsage`, call `SaveCacheAsync`, return result
         - On timeout (TaskCanceledException where ct not cancelled): retry
      7. After 3 failed attempts: return null (last valid data remains in `_cachedUsage`)

    - **Org ID migration** (when orgId is null but sessionKey exists):
      1. Call `GET https://claude.ai/api/organizations` with sessionKey cookie
      2. Parse JSON array, extract first object's `uuid` field
      3. Save via `_credentialService.SaveOrganizationId(uuid)`
      4. Return the uuid for immediate use
      5. If this also fails, return null (user needs to re-login)

    - `GetCachedUsage()`: Return `_cachedUsage`
    - `SaveCacheAsync(UsageResponse data)`: Serialize to JSON, write to cache file path. Create directory if needed. Use `File.WriteAllTextAsync`.
    - `LoadCacheAsync()`: Read cache file, deserialize, store in `_cachedUsage`, return. If file missing or corrupt, return null (no exception).

    **2. App.xaml.cs** -- Register ClaudeApiService in DI:
    - Add `services.AddSingleton<IClaudeApiService, ClaudeApiService>();` in ConfigureServices()

    - Test 401 handling: verify AuthStateChangedMessage sent, null returned
    - Test retry behavior: mock HttpClient to return 500 twice then 200 -- verify 3 attempts made
    - Test cache roundtrip: SaveCacheAsync then LoadCacheAsync returns same data
    - Test org ID migration: mock /api/organizations response, verify orgId extracted and saved
    - Note: Use a mock/fake HttpMessageHandler to intercept HTTP calls. Moq can mock ICredentialService directly.
  </action>
